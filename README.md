# LaTeX Image Auditor

A zero-dependency utility that finds and manipulates unused images in LaTeX projects.

## Installation

```bash
git clone https://github.com/konmenel/latex_image_auditor.git
cd latex_image_auditor
pip install .
```

## Running

```bash
latex-image-auditor [root]
```

or

```bash
latexaudit [root]
```

For help run

```bash
latexaudit -h
```
